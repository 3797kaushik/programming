Css Tips & Tricks to Follow

FONT - Property

font-style : normal* | italic
font-variant : normal* | small-caps
font-weight : lighter | light | bold | bolder | int (100 - 900)
font-size : (num) px
font-family : name of the font family

MEASUREMENTS - Property

m , px* , pt , em , cm , in , %

TEXT - Property

letter-spacing : (num)px
word-spacing : (num)px
text-align : left* | center | right | justify
text-decoration : none | underline | line-through | overline
text-transform : none | capitalize | uppercase | lowercase
text-indent : (num)px
line-height : (num)px
color : name of the color

BACKGROUND - Property

background-color : name of color;
background-image : url(path of image) ;
background-repeat : repeat * | no-repeat | repeat-x | repeat-y
background-attachment : scroll* | fixed
background-position : {x-axis , y-axis} or top , left , bottom, right , center

LIST - Property

list-style-type : none | disc* | circle | square | decimal | upper-alpha | upper-roman | lower-alpha | lower-roman
list-style-position : inside | outside *
list-style-image : url(path of image)

BLOCK & INLINE ELEMENTS

display : block | inline | none

VISIBILITY ELEMENTS

visibility : visible* | hidden

PADDING ELEMENTS

padding : (num)px
padding : (num)px (num)px
padding : (num)px (num)px (num)px (num)px
padding-top : (num)px
padding-right : (num)px
padding-bottom : (num)px
padding-left : (num)px

MARGIN ELEMENTS

margin : (num)px
margin : (num)px (num)px
margin : (num)px (num)px (num)px (num)px
margin-top : (num)px
margin-right : (num)px
margin-bottom : (num)px
margin-left : (num)px

BORDER ELEMENTS

border-width : (num)
border-style : solid | dashed | dotted | double | groove | ridge | inset | outset
border-color : name the color
border : width style color
border-top : width style color
border-right : width style color
border-bottom : width style color
border-left : width style color
border-radius : (num)px

OVERFLOW ELEMENTS

overflow : visible* | hidden | scroll | auto

CURSOR PROPERTY

cursor : pointer | wait

POSITION PROPERTY

position : relative | absolute | fixed

FLOAT PROPERTY

float : left | right

CLEAR PROPERTY

clear: left | right | both
