Downloading : Download of a file is done by sending few headers followed with reading of source file from the server sending a content type headersheader("Content-Type:<mine Type>");pdf -> application/pdf.html -> text/html.jpg/png/gif -> image/(jpg || png || gif).Step 2 : Sending a content Disposition header which will tells the browser to open the file or to ask the user about the file.header("Content-Disposition:<mode>;filename=<filename>");|->Inline|->attachementOptionally we can sent content length header with the size of the file.step 3 : header("Content-Length:<filesize(finename)>");Reading the Original filereadfile(<path of file>) -> Reads the file Content & prints to the browser..
